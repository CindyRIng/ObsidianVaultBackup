# 计算机网络

## 网络层

网络层负责为接收到的IP数据包寻找下一跳的地址，或者交付给网络内的主机。网络层的主要设备是路由器，实现机制是通过路由表。

### 路由表格式

| 网络前缀         | 下一跳          |
| ------------ | ------------ |
| 192.168.0.0  | 1            |
| 192.168.64.0 | 2            |
| 10.1.1.0     | 192.168.0.10 |
| 其他           | 192.168.0.15 |
流程

1. 从上层或者下层接收到一份IP数据报
2. 用IP数据报的目标IP地址分别与网络前缀
3. 从匹配后一致的网络前缀中选取最长的
4. 得到下一跳地址的物理地址，然后递交给数据链路层
